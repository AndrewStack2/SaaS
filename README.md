# Spider Fleet Management App

A real and functional SaaS web application for fleet management, built with Next.js, TypeScript, and Tailwind CSS.

## Features

- **Radar / Vehicle Tracking**: Real-time monitoring of fleet status and location.
- **Vehicle List**: Comprehensive list of all vehicles with filtering and status indicators.
- **Vehicle Detail View**: Detailed information including:
    - Overview & Health
    - Inspections History
    - Service Records
    - Documentation Management
    - Verifications & Activity Logs
- **Notification Center**: Alerts for document expirations and critical fleet events.
- **Settings & Administration**:
    - User Management & Roles
    - Audit Trails
    - Notification Preferences
    - Fleet Constants & Operational Units

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Icons**: Material Symbols Outlined & Lucide React
- **Theming**: Support for Light and Dark modes

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
